# GERMAN_Utility

## Проект Олега Германа

Этот каталог содержит набор утилит и вспомогательных библиотек, разработанных для использования на платформе MOKO SE по автоматизации измерений.

### Содержимое

*   **PythonDocs**: Python-обертки для утилит, позволяющие использовать их из Python-скриптов.
*   **UtilityCompiled_DLL**: Скомпилированная версия основной утилиты в виде DLL (`Info.dll`) и сопутствующие файлы.
*   **Utulities**: Коллекция утилит в формате LabVIEW (.vi), сгруппированных по категориям:
    *   `CSM ISRAEL`, `CSM MOSCOW`: Утилиты, связанные с конкретными проектами или заказчиками.
    *   `GERMAN`: Персональные утилиты.
    *   `MOKO`: Утилиты, специфичные для платформы MOKO SE.
    *   `Pressure Gauge`: Утилиты для работы с измерителями давления.
    *   `TLM`: Утилиты для телеметрии.
*   **Проектные файлы LabVIEW**: Исходный код и проектные файлы LabVIEW (`.lvproj`, `.lvlps`).

### Назначение

Проект `GERMAN_Utility` предоставляет набор переиспользуемых компонентов и вспомогательных функций, которые упрощают разработку и стандартизируют выполнение общих задач в рамках проектов на платформе MOKO SE.
